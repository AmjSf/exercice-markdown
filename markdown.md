## 4 Syntaxe de code
 
Pour donner une coloration de la syntaxe de code on doit entourer le code comme suit en précisant le language utilisé :
```md
```html
<body>
    <div>
        <h1>Mon Super Site !<h1>
    </div>
</body>
`` `
```
Ce qui donne :
 
```html
<body>
    <div>
        <h1>Mon Super Site !<h1>
    </div>
</body>
```
